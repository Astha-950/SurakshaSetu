# 🛡️ Suraksha Setu — Women's Support Platform

Suraksha Setu is a comprehensive women's support platform designed to provide a safe space for raising issues, sharing experiences, accessing essential helplines, and reading informative blogs. We connect women with resources, support networks, and professionals who can help navigate challenging situations and empower them to take control of their lives.

---


## ✨ Features

- **🆘 Raise Issues** — A safe and anonymous space for women to report problems, seek help, and get connected with the right support.

- **💬 Share Experiences** — A community-driven space where women can share their stories, struggles, and victories to inspire and support each other.

- **📞 Essential Helplines** — Quick access to verified helplines for emergency support, legal aid, mental health, domestic violence, and more.

- **📖 Informative Blogs** — Curated and expert-written blogs covering topics like safety, mental health, legal rights, and empowerment.

- **🤝 Support Networks** — Connect with professionals, NGOs, and communities who are ready to help.

- **🔒 Safe Space** — Built with privacy and safety in mind so every woman feels secure using the platform.

## ⚙️ Setup and Installation

### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/suraksha-setu.git
cd suraksha-setu
```

### 2. Install Dependencies
```bash
# Frontend
cd frontend
npm install

# Backend
cd backend
npm install
```

### 3. Setup Environment Variables
Create a `.env` file:
```
DATABASE_URL=your_database_url
JWT_SECRET=your_jwt_secret
PORT=5000
```

### 4. Run the Application
```bash
# Backend
npm run dev

# Frontend
npm start
```

## 🎯 Problem It Solves

Many women face challenges in:
- Finding safe and reliable support quickly

- Sharing experiences without fear of judgment
- Understanding their legal rights and options

**Suraksha Setu** solves all of this in one unified platform.
